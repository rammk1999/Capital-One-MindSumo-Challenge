S.F.P.D. Data Analysis

This web page displays my analysis and findings from a data set representing a twelve day stretch of all the 911 calls received by the San Francisco Police Department. From this data I generated 5 graphs that show various trends in the dispatches of all S.F.P.D forces. In addition, there is a dispatch prediction model.

Average Dispatch Time

This graph shows the average dispatch time for each zip code. That is, the average amount of time from when an operator receives a phone call until they can dispatch a free unit (in seconds). Most of the zip codes are pretty consistent, taking about 180 seconds to receive a call and dispatch a police unit. But in zip codes 94105 and 94127 the average times to dispatch units are so much higher than the rest of the zip codes, taking over 1500 seconds in both instances (25 minutes). In order to further flush out why this might be happening, I looked at the total number of calls each battalion took.

Total Calls per Battalion

This graph shows the the total number of calls each battalion took over the twelve day period. Most battalions responded to about 750 calls over the twelve day period. Battalion B99 seems like it is a special unit, so let's disregard it. Battalions B02 and B03 each took close to 1750 calls, approximately 1000 more calls than most other battalions. Now that the most used battalions are highlighted, let's look at where these battalions operate inside San Francisco.

Battalion Spread

This graph shows all the calls each battalion received as dots on a scatter plot consisting of the latitude and longitude of each call. Looking at the zip codes of the longest dispatch times, the battalions with the most use, and the coordinates where each battalion operates, there are two revelations. The first is that 94105 seems to have long dispatch times because there is a lot of activity in that area, and maybe the battalions don't have enough man power to respond to every call promptly. This area is also closer to downtown so it is very crowded. These two things coupled together can explain the high dispatch times. The second revelation is much more interesting. 94127 is usually assigned to battalion B08, which takes a fairly average amount of calls. So why the extraordinarily long dispatch time? After doing some further research, I discovered that there are 3 police stations in the area. The Taraval and Ingleside stations plus the SFO Police Academy. The only thing I can infer is that none of the stations are responsible for 94127 exclusively and there may be confusion on how to handle calls from this zip code. The obvious solution seems to be to either give a station full control of the area of create a new, smaller station just for this zip code.

Unit Spread

This graph is similar to the battalion spread, however each scatter plot in this facet grid represents the various unit types that the S.F.P.D sends out. By showing the distribution of each unit type, this scatter plot simultaneously where the most calls come from. Additionally, depending on the type of unit, there is also a way to determine where certain types of crimes occur. For example, the distribution of rescue squads most likely shows were very severe incidents, most likely involving hostages occur. Several other similar assumptions can be made from this graph.

Danger Heat Map

To further reinforce the assumptions that can be made from the graph of unit distributions, I created a heat map to show how dangerous different areas are in the city. There is no simple way to quantify how dangerous a situation is, so I made some simple assumptions to help aid me in identifying how "dangerous" a certain 911 call ended up being. For each call, I plotted the coordinates and then assigned the call a "danger" value. This value was calculated by taking the final call priority (either 2 or 3), the call type group (Non-Life Threatening = 1, Alarm = 2, Fire = 3, Potentially Life Threatening = 4), and the number of alarms set off by the call. Each area then is assigned a danger level based on the total danger of each call in that zone. The darker the zone, the more dangerous it is.
